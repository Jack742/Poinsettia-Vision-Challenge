## About The Project
<p align="center">

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/Jack742/Poinsettia-Vision-Challenge.git
   ```
2. Install Pytorch packages
   ```sh
   pip install torch torchvision
   ```
3. Install Opencv packages
   ```sh
   pip install opencv-python
   ```
